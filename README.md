# Bartosz Titkin

Environment Artist and tool developer building practical tools for artists, AI workflows and game production.

I work around ComfyUI, Unreal Engine, MCP servers, texture/material pipelines and local-first developer automation.

## What I Build

- Artist-facing tools for texture, trimsheet, UV and asset-pipeline work.
- ComfyUI nodes and local image-processing workflows for visual iteration.
- MCP servers and automation utilities for longer-running AI-assisted development.
- Experimental Unreal Engine and game-art pipeline tools.

## Featured Projects

| Project | What it does | For | Status |
| --- | --- | --- | --- |
| [OwlynMCP](https://github.com/btitkin/OwlynMCP) | Local-first MCP work-session supervisor with checkpoints, deadlines and continuation decisions. | AI coding agents and developer automation workflows. | Alpha |
| [ComfyUI-KSampler-Matrix-Lab](https://github.com/btitkin/ComfyUI-KSampler-Matrix-Lab) | ComfyUI custom nodes for sampler, scheduler and model comparison grids. | ComfyUI users testing generation settings and model behavior. | Usable |
| [TrimSheetArchitect](https://github.com/btitkin/TrimSheetArchitect) | Browser-based trimsheet layout planner for pixel-accurate texture planning. | Environment artists and technical artists. | Usable |
| [Upscale-Engine-CC](https://github.com/btitkin/Upscale-Engine-CC) | Local desktop upscaler/enhancement app built around a React/Electron frontend and Python backend. | Artists evaluating local image and texture enhancement workflows. | Alpha |
| [UVfromView](https://github.com/btitkin/UVfromView) | Blender add-on for view/global-axis based UV projection with texel-density controls. | Blender users, hard-surface artists and trim-sheet workflows. | Experimental |

## Current Focus

- Practical ComfyUI tools for comparing models, samplers and visual outputs.
- Local-first agent tooling with MCP.
- Texture and material workflow utilities for environment art.
- Automation that supports artists without hiding the underlying pipeline.

## Tech Stack

ComfyUI, Python, TypeScript, React, Vite, Electron, MCP, SQLite, Blender add-ons, Unreal Engine pipelines, texture/material workflows.
